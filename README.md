# Loure Coffee Shop

Web aplikasi coffee shop berbasis Next.js dengan fitur lengkap untuk manajemen pesanan, pembayaran QRIS, dan admin panel.

## Fitur

### Untuk Pelanggan

- Lihat menu dengan rating dan harga
- Registrasi dan login dengan OTP Email
- Tambah ke keranjang dan checkout
- Pembayaran QRIS (GoPay, ShopeePay, dll)
- Tracking pesanan real-time
- Edit profil

### Untuk Admin

- Dashboard dengan statistik penjualan
- Kelola menu (CRUD)
- Riwayat pesanan dengan filter tanggal
- Antrian pesanan real-time
- Kasir manual
- Laporan harian/bulanan (Export Excel)
- Kelola admin dengan log aktivitas

## Teknologi

- Next.js 15 (App Router)
- TypeScript
- Tailwind CSS 4
- DaisyUI 5
- Firebase (Firestore, Auth, Storage)
- EmailJS (OTP)
- Midtrans (Pembayaran QRIS)
- XLSX (Export Excel)
